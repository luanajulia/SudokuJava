# 🧩 Sudoku Master - Java Edition

Um jogo de Sudoku clássico desenvolvido inteiramente em **Java** para fortalecimento de conceitos de lógica de programação, algoritmos de busca e estruturas de dados multidimensionais.

---

## 🎮 O Projeto
Este projeto foi criado com o objetivo de aplicar conhecimentos puros de Java, focando na implementação de regras de validação lógica e na experiência do usuário via console (ou interface gráfica, caso tenha usado Swing/JavaFX).

## ✨ Funcionalidades
- [x] **Geração de Tabuleiro:** Criação de matrizes 9x9 seguindo as regras oficiais.
- [x] **Validação em Tempo Real:** Verifica se o número inserido respeita a regra da linha, coluna e do quadrante 3x3.
- [x] **Verificação de Vitória:** Identifica quando o desafio foi concluído com sucesso.
- [x] **Algoritmo de Resolução (Opcional):** Implementação de Backtracking para resolver o tabuleiro automaticamente.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** [Java](https://www.oracle.com/java/) (JDK 17+)
* **Paradigma:** Programação Orientada a Objetos (POO)

## 🧠 Conceitos Aplicados
Este desafio me permitiu aprofundar em:
1. **Matrizes (Arrays Bidimensionais):** Manipulação de coordenadas `[i][j]`.
2. **Recursividade:** Uso de funções recursivas para validação e busca.
3. **Tratamento de Exceções:** Garantir que o usuário insira apenas números válidos.
4. **Lógica Booleana:** Verificações complexas para as regras do jogo.
